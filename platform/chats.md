# Chats

The Chats section provides a direct interface for interacting with your AI assistants. This feature allows you to test assistant capabilities, train assistants through conversation, and demonstrate assistant functionality to stakeholders.

## Overview

The Chats page is organized into a two-panel layout:

1. **Sessions List** - Browse and manage your chat sessions
2. **Chat Interface** - Interact directly with the selected assistant

## Features

### Sessions List

The Sessions List panel displays your existing chat sessions:

- Each session shows:
  - Assistant name and avatar
  - Session start time/date
  - Preview of the last message
  - Unread message indicator (if applicable)

To manage your chat sessions:

1. Click on a session to open it in the chat interface
2. Use the search bar to find specific sessions
3. Create a new session by clicking the "New Chat" button
4. Delete sessions you no longer need

### Chat Interface

The Chat Interface panel is where you directly interact with your assistants:

- Message history displayed in a conversation format
- Input field for typing new messages
- Real-time responses from your assistant
- Support for various message formats (text, images, links)
- Message status indicators (sent, delivered, read)

Key features of the chat interface include:

1. **Real-time Communication**:
   - Messages are sent and received in real-time
   - Typing indicators show when the assistant is preparing a response

2. **Rich Media Support**:
   - Share images and files with your assistant
   - Receive formatted responses with links and structured content
   - View and interact with embedded content

3. **Voice Interaction**:
   - Text-to-speech capability for assistant responses
   - Voice input option for sending messages

4. **Context Awareness**:
   - Assistants maintain context throughout the conversation
   - Reference previous messages in the same session

### Assistant Selection

When starting a new chat:

1. Click the "New Chat" button
2. Select an assistant from your available options
3. Begin your conversation with the selected assistant

You can switch between different assistants to compare their responses or test specific capabilities.

### Chat History

Your chat history is preserved for future reference:

- Scroll through previous messages within a session
- Search for specific content within the current chat
- Export conversation logs for documentation or training purposes

## Best Practices

- Use the Chats feature to test your assistants before deploying them to users
- Create separate chat sessions for different topics or use cases
- Review chat history to identify areas where assistant responses could be improved
- Use real-world scenarios in your test conversations to ensure assistants can handle typical user queries
- Regularly clear old or unnecessary chat sessions to keep your workspace organized

## Technical Notes

- Chat sessions use WebSocket connections for real-time communication
- Messages are securely transmitted and stored
- The interface automatically adjusts to different screen sizes with resizable panels
