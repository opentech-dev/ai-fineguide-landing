# Assistant > Features Tab

## Overview
The **Features** tab allows users to configure various functionalities and behavioral settings of their AI assistants. This tab focuses on customizing user interaction, response handling, lead collection, and ticket management.

### 1. **Show Suggested Questions**
This section allows users to configure predefined questions that the assistant can suggest to users during interactions. 

- **Suggested Questions Toggle:** A switch to enable or disable suggested questions functionality.
- **Existing Suggested Questions:** Displayed as clickable buttons for easy access. Examples include:
  - `What is Fineguide?`
  - `How can you help my business?`
  - `Schedule a Call`
- **Add Question Input:** A text box where users can input a new suggested question.
- **Add Question Button:** A button to save the new question and add it to the list of suggested questions.

### 2. **Response Limitations**
Defines the scope of the assistantâ€™s responses, particularly in relation to context-aware interactions.

- **Restrict Assistant Toggle:** A switch that, when enabled, restricts the assistant to only answer questions related to a predefined context. This helps ensure focused and relevant responses, preventing the assistant from straying into unrelated topics.

### 3. **Lead Collection**
Settings related to collecting user information and handling lead generation through conversations.

- **Collect Customer Contacts Toggle:** A switch to enable or disable lead collection functionality. When enabled, the assistant will actively collect contact details during interactions.
- **Lead Collection Mode:** Dropdown menu providing different modes of handling lead generation. Options include:
  - `Allow conversations, ask lead details on each reply`: The assistant engages in natural conversations and prompts the user for contact details whenever appropriate.
  - `Require contact details before conversation`: The assistant requests contact information before proceeding with interactions.
  - `No lead collection`: Disables lead collection entirely.

### 4. **Tickets**
Options related to ticket generation and automatic issue detection.

- **Allow Ticket Creation Toggle:** Allows users to create support tickets directly from the chat menu. This makes it convenient for users to report issues or request support.
- **Automatic Ticket Creation Toggle:** Enables the assistant to automatically detect issues or requests and create tickets as needed. This can be particularly useful for customer support scenarios where the assistant is expected to streamline the issue resolution process.

## Usage & Purpose
The **Features** tab is designed to enhance user interaction by providing customized suggestions, refining response behavior, and enabling structured lead collection. Additionally, it allows for seamless ticket management through automated detection and user-initiated ticket creation.
