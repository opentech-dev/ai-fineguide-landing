# Assistant > Appearance Tab

## Overview
The **Appearance** tab allows users to customize the visual presentation of the assistant’s chat window and widget that users embed on their websites. This includes themes, widget positioning, branding options, and other visual adjustments to ensure a seamless fit with the website's aesthetic.

### 1. **Theme Customization**
This section allows users to configure the **Light Theme** and **Dark Theme** of the chat window.

- **Color Settings:**
  - **Button Color:** Adjusts the color of buttons within the chat interface.
  - **Text Color:** Controls the text color for chat messages and interface elements.
  - **Background Color:** Sets the chat background color.
  - **Borders:** Controls the border color and styling.
  - **Font Color:** Adjusts text font color within the chat.
  - **Highlight/Action Color:** Specifies the color used for highlighted elements or call-to-action buttons.
  - **Effect Colors:** Manages accent or animation effects applied within the chat interface.
  
- **Reset Styles Button:** Resets all customization settings to their default values.
- **Save Values Button:** Saves the current customization settings for future use.

### 2. **Chat Window Configuration**
- **Chat Interface Language:** Dropdown menu allowing users to select the language displayed within the chat interface.
- **Chat Template:** Allows users to select predefined or custom templates for the chat layout.
- **Chat Header:** Option to toggle the visibility of the chat window header.
- **“Hide Close Chat” Button Toggle:** Allows the user to hide or display the close button in the chat interface.
- **Fineguide Footer Branding Toggle:** Toggle to enable or disable Fineguide’s branding in the footer of the chat interface.
- **Fineguide Footer Positioning:** Option to select the branding’s position (e.g., bottom left, bottom right, etc.).

### 3. **Widget Configuration**
This section deals with the configuration of the assistant’s icon widget that is displayed on the website.

- **Widget Effect:** Dropdown to choose animations or effects applied to the widget icon (e.g., `Expanding Ring`).
- **Widget Position:** Allows users to position the widget icon on their website’s interface. Options include:
  - `Top Left`, `Top Center`, `Top Right`
  - `Center Left`, `Center Right`
  - `Bottom Left`, `Bottom Center`, `Bottom Right`
- **Widget Bubble Message:** Input field to customize the tooltip message that appears when users hover over the widget icon.
- **Widget Size:** Adjustable slider to change the size of the widget icon. The slider range typically goes from small to large.

### 4. **Preview & Testing**
- Users can preview their changes directly on this page. The widget and chat interface on the right side of the screen will reflect updates in real-time, providing a live preview of appearance adjustments.

## Usage & Purpose
The **Appearance** tab is designed to give users full control over how their assistant appears on their website. By customizing colors, layout, positioning, and branding, users can ensure the assistant fits seamlessly within their existing web design and enhances user experience.
