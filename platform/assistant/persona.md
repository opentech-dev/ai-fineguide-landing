# Assistant > Persona tab

## Overview
The **Assistant Persona** page allows users to configure and customize individual AI agents within the Fineguide.ai platform. This initial tab is focused on defining the general characteristics, personality traits, and settings of a specific assistant. 

### 1. **General Settings**
This section provides basic details and customization options for the assistant.

- **Name:** The name of the assistant (e.g., Buddy). This name is user-defined and can be modified to suit individual use cases.
- **Company or Owner:** The name of the entity or business associated with the assistant (e.g., Fineguide).
- **LLM Model:** Dropdown selection for choosing the underlying language model powering the assistant. Example: `GPT-4`.
- **Voice Model:** Dropdown selection for selecting a voice model for audio interactions (e.g., `Charlie`).
- **Role:** The designated purpose or functionality of the assistant, such as `Customer Support`, `Sales Agent`, etc.
- **Welcome Message:** The initial greeting message displayed by the assistant to the user. Example: `Hi, I'm here to assist you with Fineguide.`
- **Notes:** An optional text field where users can leave internal notes related to the assistant’s purpose or functionality (e.g., `Homepage Bot`).

### 2. **Personality Settings**
This section allows users to define the communication style and behavior of the assistant.

- **Manner of Speaking:** Defines the tone and style of the assistant’s speech. Options include:
- `Normal`
- `Friendly`
- `Formal`
- `Concise`
- **Persuasiveness:** Specifies how assertive or suggestive the assistant is when delivering messages. Options include:
- `Assertive`
- `Neutral`
- `Passive`
- **Verbosity:** Controls the level of detail provided by the assistant in its responses. Options include:
- `Normal`
- `Verbose`
- `Concise`
- **Follow-up Questions:** Toggle switch enabling or disabling the assistant’s ability to ask follow-up questions for better clarity or engagement.

### 3. **Custom Prompts**
An expandable section allowing users to define and customize prompts that the assistant can use. This can enhance the assistant's performance by tailoring responses to specific scenarios or inquiries.

### 4. **Danger Zone**
A section dedicated to deleting the assistant permanently. This action is irreversible and requires user confirmation. 
- **Delete Button:** Displays a prominent red button labeled `Delete [Assistant Name] Permanently`.

### 5. **Chat Simulation Panel (Right Side)**
A live simulation panel that demonstrates how the assistant will interact with users based on the configurations set. This allows for testing and previewing responses in real-time.
